mergeDb By Sawyer Shipp-Wiedersprecher
for Kbotics 2013 scouting database


1. Place both scouting.db files in 'file1' and 'file2' folders
2. Run the program once
3. The information in file2 will be injected into file1 
4. Remove scouting.db from file1, you may delete file2 unless it is otherwise required
